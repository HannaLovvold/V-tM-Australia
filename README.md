# Vampire: The Masquerade - V5 Resource Kit

A comprehensive collection of resources, tools, and materials for Vampire: The Masquerade 5th Edition (V5) tabletop roleplaying sessions.

## Overview

This repository contains game master resources, NPC profiles, city settings, lorebooks, and safety tools for running V:tM V5 chronicles. All materials are designed to enhance storytelling and streamline game preparation.

## About Vampire: The Masquerade

Vampire: The Masquerade is a tabletop roleplaying game published by Paradox Interactive (formerly White Wolf Publishing). It is set in the *World of Darkness* - a gothic-punk reflection of our world where vampires, werewolves, and other supernatural creatures exist among humans.

### Game Edition
- **Vampire: The Masquerade 5th Edition (V5)** - Current edition
